ACA Canvas Users
================

A Django application for adding users to Canvas courses aligned with UW policy

Installation
------------

**Project directory**

Install django-canvas-users in your project.

    $ cd [project]
    $ pip install -e git+https://github.com/uw-it-aca/django-canvas-users/#egg=django_canvas_users

Project settings.py
------------------

Just add "canvas_user" to INSTALLED_APPS
